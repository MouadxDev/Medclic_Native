## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Expo CLI
- Android Studio or Xcode (for native development)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd medclic
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

### Running the App

- For Android: `npm run android`
- For iOS: `npm run ios`
- For Web: `npm run web`
